Machine Learning is the process of allowing automation to make predictions on datas and in turn provide solutions
to many of the modern day problems.Machine learming is a subset of Artificial Intelligence(AI) the machine is allowed to learn by providing as
much data possible, we train the model with specific algorithms without explictly coding.This kind of models are mainly used i the area of social media,
marketing,risk detection and even more.

Machine learning can be mainly classified as :
  1. Supervised Learning
  2. Unsupervised Learning
  3. Reinforced Learning

Supervised Learning

This type of machine learning is when we deal with labelled data like table.It includes problems that require:

  • Regression
  • Classification
Under the regresson there are types like linear and polynomial.eg. predicting slary of a employee. Classification models are KNN,Naive bayes,SVM etc. about which you will learn deep later.

Unsupervised Learning

Here we deal with the unlabelled (unstructered data) like images,audio,video etc.This kind include finding out the
similarities an dissimilarities by itself and learning.It includes clustering problems.eg. for unsupervised learning is clustering based
on the amount of money spent by a person,face recognition,speech detection etc.

Reinforced Learning

the kind of learning where u do not provide any data but still the machine learning. This is by using the reward and
punishment cycle.The machine is designed to perform a task and every time it does a correct thing it is rewarded and if it goes wrong the
machine recieves this as punishment and correct this and learns from this to perform better the next time.. eg Self Driving Cars.

Beginners generally practise more supervised then unservised followed by reinforced.
